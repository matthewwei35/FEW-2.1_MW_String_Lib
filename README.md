# String Library
 
## Description
This library contains functions used to manipulate strings.

## Functions
```javascript
upperFirst(str)

allCaps(str)

capitalizeWords(str)

removeExtraSpace(str)

kebabCase(str)

snakeCase(str)

camelCase(str)

shift(str)

makeHashTag(str)

isEmpty(str)
```

### Helper Functions
```javascript
lowerFirst(str)

removeItem(array, item)
```
